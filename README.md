# Cpiupiu
#### Do you like C++? Do you like Italian? Include this header file and create your C++ program in italian!

### Installation
Download cpp_ita.h file from this repository and include in your project with
```c++
#include "./cpp_ita.h"
```

### Example
```c++
#include <iostream>
#include "./cpp_ita.h"
using namespace std;

intero principale() {

    stampa << "test";

    ritorna 0;
}
```
